Rho.WebView

Rho.WebView.framework:
[VT301-023]
Browser framework identity string.
Rho.WebView.activeTab:
[VT301-004 / VT301-005]
An active tab index.
Rho.WebView.fullScreen:

[VT301-024/VT301-027/VT301-025]
Use full screen mode.
Rho.WebView.enableCache:

[VT301-008/VT301-009]
Enable/disable Browser cache. Not yet implemented.
Rho.WebView.currentLocation():


[VT301-053/VT301-054/VT301-055]
Returns the url (location) of the current page; the last URL loaded to WebView.
Rho.WebView.navigate():


[VT301-063/VT301-064/VT301-067]
Force WebView to navigate to a URL.
Rho.WebView.navigateBack():





[VT301-063/VT301-064/VT301-067]
Force WebView to navigate to the previous page using Browser back.
Rho.WebView.refresh():



[VT301-072/VT301-073/VT301-074/VT301-075]
Force WebView to refresh the current page.

Enable WebView zoom.
Rho.WebView.enableWebPlugins:

[VT301-015/VT301-016]
Set enabling web plugins
Rho.WebView.save():



[VT301-076/VT301-077/VT301-078/VT301-079]
Save current page to file system.

Non WM specific methods

Rho.WebView.setCookie:


[VT301-080/VT301-081]
When WebView loads the specified url (either by selecting link or from calling WebView.navigate), it will add this cookie to the HTTP request.

WM specific methods

Rho.WebView.scrollTechnique:
[VT301-028/VT301-029/VT301-030]
Specifies the technique used to scroll about the page. Defines in config.xml: Scrolling\ScrollTechnique.
Rho.WebView.fontFamily:
[VT301-021/VT301-022]
Specifies the default font to use when rendering text in web pages. Defines in config.xml: HTMLStyles\FontFamily.
Rho.WebView.userAgent:
[VT301-031/VT301-032]
When visiting a web server the WebKit browser will report its self as the specified user agent. Defines in config.xml: Navigation\UserAgent.
Rho.WebView.viewportEnabled:
[VT301-033/VT301-034]
Whether to enable or disable viewport meta tag processing. Defines in config.xml: Navigation\ViewportEnabled.
Rho.WebView.viewportWidth:
[VT301-035/VT301-036]
Default viewport width to use for pages that do not have a viewport meta tag (uses 1:1 scaling if not specified). Defines in config.xml: Navigation\ViewportWidth.
Rho.WebView.cacheSize:
[VT301-006 / VT301-007]
Browser cache size, in whole MBs. Defines in config.xml: Navigation\Cache.
Rho.WebView.navigationTimeout:

[VT301-045/VT301-050]
Number of milliseconds before the browser times out and navigates to the page specified in the badlink setting. Defines in config.xml: Navigation\NavTimeout.
Rho.WebView.enablePageLoadingIndication:

[VT301-011/VT301-013]
Show page loading navigation. Defines in config.xml: GUI\HourglassEnabled
Rho.WebView.acceptLanguage:


[VT301-001/VT301-002/VT301-003]
Configurable HTTP headers. Not yet implemented.
Rho.WebView.zoomPage:



[VT301-037/VT301-038/VT301-039/VT301-040]
Sets the zoom factor of the page. Factor 1.0 is no zoom, values less than 1.0 are zoomed out and values greater than 1.0 are zoomed in.
Rho.WebView.textZoomLevel:



[VT301-041/VT301-042/VT301-043/VT301-044]
Sets the font size to be displayed on the page, set to 0 for the smallest font and 4 for the largest font.
