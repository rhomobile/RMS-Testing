About (Top)
This screen shows the versioning information for BTExplorer. More detailed version information can be found by clicking the version button.
A2DP Connection (Top)
This dialog allows you to make a connection to remote A2DP Sink Service.
AVRCP Connection (Top)
This dialog allows you to make a connection to remote AVRCP Controller Service.
Setup Connection (Top)
Connection Setup defines what hardware is used for communication with the Bluetooth chipset.

Physical Device Connection indicates which COM or USB port is used.

Baud Rate indicates the speed at which a COM port communicates.

Protocol indicates the protocol at which a COM port communicates at.


Virtual COM Port (Top)
Virtual COM Port defines which COM ports BTExplorer will attempt to use for virtual COM ports. Check the appropriate checkbox to use the port as a virtual COM port. When you are finished, you can choose "Apply" to enforce your changes, or "Revert" to restore the original settings.
Setup Discovery (Top)
Discovery allows you set Bluetooth discovery settings.

Inquiry Length Length of time that BTExplorer performs a device inquiry. An inquiry is used to find other Bluetooth devices.

Name Discovery Mode Whether or not BTExplorer automatically attempts to discover a Bluetooth device's name after finding the device.

Delete Devices Deletes all existing remote Bluetooth devices from BTExplorer's memory.

Delete Link Keys Removes all pairing from remote Bluetooth devices, and makes them all untrusted.


Device Info (Top)
Device Info allows you to set information for your device.

Device Name Name of this device that is displayed to other Bluetooth devices.

Discoverable Mode Whether or not the device can be discovered by other Bluetooth devices.

Connectable Mode Whether or not the device accepts connections from other Bluetooth devices.


Setup Security (Top)
Security Settings allows you to set global security policies for Bluetooth. Note: These settings are ONLY active on local Services that are set to Authenticate or Authenticate/Encryption. You can set authentication on local Services under the Services tab.

Use PIN Code Checking this forces both your local device and the remote device to authenticate each other by entering a PIN code before communicating. If you wish, you can also enter a pre-created PIN code in the box below. (If there is no PIN code in the box, both devices will be prompted to enter a PIN code when the connection is attempted.)

Encrypt Link Checking this forces encryption on all outgoing connections to other Bluetooth devices.


Setup Roles (Top)
This screen shows device settings for Bluetooth's L2CAP link configuration parameters.

Disallow/Allow Role Switch allows the L2CAP layer to be set to disallow or allow role switching.

Maintain Current Role/Request Role Switch allows the L2CAP layer to be set to accept new role or maintain an existing role.


Setup Services (Top)
This screen shows various Services on this device which other devices may connect to. With this panel, you can add, delete, and modify services that are offered by this local device. To add a service, choose "Add". To delete a service, choose the service to delete, and choose "Delete". To change properties of an existing service, choose the service, and choose "Properties".
Inquiry (Top)
BTExplorer is currently querying all remote Bluetooth devices to see which devices are available. The progress bar shows when this query will be complete. You can cancel this action by clicking the Cancel button.


Name Discovery (Top)
BTExplorer is currently querying all remote Bluetooth devices for their names. The progress bar shows when this query will be complete. You can cancel this action by clicking the Cancel button.


Pairing (Top)
Your local device is attempting to pair with a remote device.

Device Address lists the Bluetooth Address that the remote device has.

Device Name lists the name of the remote device that you are attempting to pair with.

Cancel allows you to cancel the pairing attempt.


PIN Code (Top)
A remote device is asking you for an authentication PIN code. Enter the PIN code and click "OK". If you enter the correct PIN code, your device will be considered paired or "trusted" by the remote device.

Note: Unless this device is a keyboard, this PIN code comes from the remote device. Consult the device's documentation or speak with the owner of the device to get the PIN code. If this device is a keyboard, enter a random pin code and then enter the same pin on the keyboard and press Enter (or Return). For security, the keyboard does not echo the PIN code entered.


Remote Service Connection (Top)
This dialog allows you to make a connection to a Serial Port or Active Sync Service.

Address lists the Bluetooth Address that the remote device has.

Device Name lists the name of the remote device that you are connecting to.

Service Name lists that this is a Serial Port or Active Sync Service. If you are attempting to Active Sync with a device, you can choose either Active Sync or Serial Port in this selection.

COM Port lists which COM: port on this device will be used for the Serial Port connection.

Baud Rate lists the baud rate that the device will use when transferring data.

Port Options lists the port options that the device will use when transferring data.


HCRP Connect (Top)
This dialog allows you to make a connection to a Hardcopy Cable Replacement (printing) Service.

Address lists the Bluetooth Address that the remote device has.

Device Name lists the name of the remote device that you are connecting to.

Service Name lists that this is a Hardcopy Cable Replacement Service.

COM Port lists which COM: port on this device will be used for the Hardcopy Cable Replacement connection.

Baud Rate lists the baud rate that the device will use when transferring data.

Port Options lists the port options that the device will use when transferring data.


File Transfer (Top)
In this screen, you can watch the current status of (and cancel) the progress of the File Transfer. The status bar on the bottom of the screen shows the current progress of the file transfer.

Device Address lists the Bluetooth Address that the remote device has.

Device Name lists the name of the remote device that you are transferring the file with.

Service Name lists that this is an File Transfer Service.

File Name lists the name of the file that is being transferred.

Cancel Hitting this button will stop the file transfer.


OBEX Object Push (Top)
In this screen, you can watch the current status of (and cancel) the progress of the OBEX Object Push. The status bar on the bottom of the screen shows the current progress of the object push.

Device Address lists the Bluetooth Address that the remote device has.

Device Name lists the name of the remote device that you are pushing the object to.

Service Name lists that this is an OBEX Object Push Service.

Cancel Hitting this button will stop the object transfer.


Add Profile (Top)
In this screen, you can choose local Services which will be available to other Bluetooth devices. Choose a Service in the list which you would like to run on this device (such as the File Transfer Service) by clicking on it, then click OK.
Create/Edit A2DP Service (Top)
In this screen, you can edit the settings for an Advanced Audio Distribution Profile (A2DP) Service.

Service Name lists the displayed name of the A2DP Service.


Create/Edit AVRCP Service (Top)
In this screen, you can edit the settings for an Audio Video Remote Control Profile (AVRCP) Service.

Service Name lists the displayed name of the AVRCP Service.


Create/Edit Serial Port (Top)
In this screen, you can edit the settings for a Serial Port Service. A Serial Port Service mimics the functionality of a standard serial port, but can pass data over Bluetooth. This Service is also used by Dial-Up Networking.

Service Name lists the displayed name of the Service.

Service Security indicates what level of security is to be used for the Service. Settings are None (which indicates no security), Authenticate (which forces connecting devices to enter a PIN code), and Authenticate/Encrypt (which forces connecting devices to enter a PIN code and also encrypts communications between the two devices).

COM Port lists which COM: port on the device will be used by the Service.

Baud Rate lists the baud rate that the Service will transfer data at.

Port Options lists the port options that the Service will use when transferring data.
Create/Edit Audio
(Top)

In this screen, you can edit the settings for an Audio (Headset, Hands-Free) Service. These services allow connections to Headset and Hands-Free devices.

Service Name lists the displayed name of the Audio Service.


Create/Edit File Transfer (Top)
In this screen, you can edit the settings for a local File Transfer Service.

Service Name lists the displayed name of the File Transfer Service.

Service Security indicates what level of security is to be used for File Transfer. Settings are None (which indicates no security), Authenticate (which forces connecting devices to enter a PIN code), and Authenticate/Encrypt (which forces connecting devices to enter a PIN code and also encrypts communications between the two devices).

Root Directory lists where on your device that other Bluetooth devices send files to.

File Permissions are a series of checkboxes that indicate what other Bluetooth devices can do to files on your device. If "Read" is checked, other Bluetooth devices can read files on your device. If "Write" is checked, other Bluetooth devices can write to your device. If "Delete" is checked, other Bluetooth devices can delete files on your device.


Create/Edit Sync Server (Top)
In this screen, you can edit the settings for a local IrMC Sync Server Service.

Service Name lists the displayed name of the Sync Server Service.

Service Security indicates what level of security is to be used for connections to the Sync Server. Settings are None (which indicates no security), Authenticate (which forces connecting devices to enter a PIN code), and Authenticate/Encrypt (which forces connecting devices to enter a PIN code and also encrypts communications between the two devices).

Server Permissions are a series of checkboxes that indicate what other Bluetooth devices can do to objects (Contacts, etc) on your device. If "Read" is checked, other Bluetooth devices can read entries on your device. If "Write" is checked, other Bluetooth devices can write to your device. If "Create" is checked, other Bluetooth devices can create entries on your device. If "Delete" is checked, other Bluetooth devices can delete entries on your device.


Create/Edit OBEX Object Push Profile (Top)
In this screen, you can edit the settings for a local OBEX Object Push Service.

Service Name lists the displayed name of the OBEX Object Push Service.

Service Security indicates what level of security is to be used for OBEX Object Push. Settings are None (which indicates no security), Authenticate (which forces connecting devices to enter a PIN code), and Authenticate/Encrypt (which forces connecting devices to enter a PIN code and also encrypts communications between the two devices). Note that there is also a checkbox which prevents clients from pushing objects to your local device.

Inbox Directory lists where on your device that other Bluetooth files push objects to.


Create/Edit PAN Profile (Top)
In this screen, you can edit the settings for a local Personal Area Networking (PAN) Service.

Service Name lists the displayed name of the Personal Area Networking Service.

Service Security indicates what level of security is to be used for Personal Area Networking. Settings are None (which indicates no security), Authenticate (which forces connecting devices to enter a PIN code), and Authenticate/Encrypt (which forces connecting devices to enter a PIN code and also encrypts communications between the two devices).

Support Group Ad-hoc Networking Check this to indicate that Personal Area Networking, Group Ad-hoc Networking (PAN-GN) is enabled for the Personal Area Networking Service. If this is unchecked, PAN-GN will not be available. PAN-GN allows additional network routing between different items in a Personal Area Network.

Remote Device Properties (Top)
List of properties for a particular remote device.

Device Address lists the Bluetooth Address that the remote device has.

Local Name lists the name of the remote device. This can be changed by the local device.

Device Name lists the name of the remote device. This cannot be changed by the local device.

Major Device Class lists the Bluetooth Device Class that this device falls into.

RSSI lists the signal strength of the connection between the local device and this remote device. The range of the signal is from -127 to 128, where 0 is considered the "strongest" signal. As the RSSI value gets larger or smaller than 0, the connection strength decreases.

Connection Status lists the current connection status between the local device and this remote device.

Paired Status lists the paired/trusted relationship between the local device and this remote device.

Encryption Status lists the encryption relationship between the local device and this remote device.


Favorite Property (Top)
List of properties for a particular Favorite.

A stored Favorite consists of two things: a remote device, and a service on that device.

Favorite Name lists the name of the Favorite. This can be changed by the local device.

Device Address lists the Bluetooth Address of the remote device that the Favorite uses.

Device Name lists the name of the remote device that this Favorite uses.

Service Name lists the name of the Service that this Favorite uses.

Service Type lists the type of the Service that this Favorite uses.

Profile Status lists the current connection status between the local device and this Favorite.

Profile Version lists the version of the Service drivers.

Channel lists the L2CAP channel, L2CAP control channel, or RFCOMM server port that this Favorite uses while connected.

This screen allows you to select which phone number to dial. You can either choose an entry by clicking on it, then choose "OK", or you can add a new entry. To add a new entry, click on an open spot in the book to bring up the popup menu, then choose "Add Entry".

This screen allows you to view and change phone book entries.

Name Name of the connection.

Select File (Top)
This screen allows you to select which file to transfer from the remote device. You can browse the remote device by clicking on folders in the listing at the bottom of the screen. When you find the file you wish to transfer, click on it, then click the "Save" button.
Setup Miscellaneous (Top)
Miscellaneous lets you set colors and types to better view active Bluetooth connections.

Highlight Connections indicates how active Bluetooth are highlighted, and depends on which mode BTExplorer is currently running in.

In Explorer mode, "None" shows no highlighting. "Tree View Only" will highlight connections only in the tree view.

"List View Only" will highlight connections only in the list view. "Tree and List View" will highlight connections in the tree and list views.

In Wizard mode, "None" shows no highlighting. "Favorites" will highlight favorites which are currently connected.<\P>

Apply Text Style indicates how the connection text is highlighted. Options are Bold, Italic, and Underline.

Apply Text Color indicates what the connection text color is.

This screen allows you rename a device. Enter in the device name and click "OK".

This screen allows you to enter a name for a new file or new folder. Enter in the name and click "OK".
OBEX Object Push Connect (Top)
This screen displays OBEX Object Push information. To send an object, choose an item under "Action" (such as "Send A Picture"), then select an object to send to the remote device (such as a JPEG image). To select an object, click on the "..." button.

Address lists the Bluetooth address of the remote device that you are pushing the object to.

Device Name lists the Bluetooth name of the remote device that you are pushing the object to.

Service Name lists that this is an OBEX Object Push Service.

Select Contact (Top)
This screen displays all the contacts you have on your local device. You can choose a contact from the list, and click OK. If you wish to create a new contact, click on an empty square to bring up the popup menu, and choose "Add Contact".
View Active Connections (Top)
This screen displays all the active connections that this device currently has to other Bluetooth devices. To disconnect a connection, click on the device, and choose "Disconnect."
View Trusted Devices (Top)
This screen displays remote Bluetooth devices that you are paired with and "trust." To remove trust from a particular device, select the device and choose "Delete Link Key" from the popup menu. Alternately, to remove trust from all devices, select anywhere on the screen and choose "Delete All Link Keys" from the popup menu.
New Connection Wizard (Top)
This screen displays actions you can perform with other Bluetooth devices. Select an action from the list and choose "Next" to begin.
Select Remote Device (Top)
This screen displays remote Bluetooth devices to connect with. Select one of these devices and click "Next". This list can be filtered using the toolbar.

If you do not see the remote device you wish to connect with, try to discover it. Click on the white background to bring up the popup menu, and choose "Discover Devices" or "Refresh".


Select Remote Profile (Top)
This screen displays Services on the remote Bluetooth device. Select a Bluetooth Service and click "Next".

If you do not see the Service you wish to connect with, click on the white background to bring up the popup menu, and choose "Refresh".

Wizard - File Transfer (Top)
This screen lets you perform File Transfers. To begin, choose the "FTP Actions" button at the top of the screen. Depending on the security settings of the remote device you are connected to, you can perform the following actions:

New Folder / New File creates a new folder or a new file on the remote device.

Delete deletes a file or folder on the remote device.

Get File transfers a file from the remote device to your local device.

Put File transfers a file from your local device to the remote device.

Parent Directory moves up a single directory level.

Refresh refreshes the current directory on the remote device.

You can also move between directories by clicking on them, or by selecting the Directory tab at the top of the screen. A "Sort By" option is available at the top right of the screen as well.


Explorer View (Top)
Explorer mode is an advanced view that can be used to quickly explore remote devices and add services to the local device. Most features are accessed by the context menus on various items such as "Local Device" and "Remote Device".

Choosing View/Wizard Mode will return BTExplorer to the normal Wizard view.


Wizard View (Top)
Wizard mode of BTExplorer shows all of the Favorite devices and services.

To create a new Favorite, click on the Favorite and select "Connect". To create a new Favorite, choose File/New Connection.


Select Association/Favorite (Top)

Choose an existing Favorite to connect to this serial port.


File Versions (Top)
This screen shows version information for each of the components upon which BTExplorer relies.
Incoming Call (Top)

This screen indicates that there is an incoming phone call coming from Hands-Free. Click OK to accept the call, or Cancel to reject the call.

Caller ID indicates any caller identification from the incoming phone call.

Ringing will flash in BOLD each time a ring is received from the incoming phone call.


Dial A Number (Top)

You can dial a phone number with this screen. Click on the digits to dial the phone number. You will see the phone number appear on the top of the screen.

To dial the number, choose Dial.

To exit without dialing, choose Cancel.


DTMF Dial (Top)

You can instantly dial DTMF (Dual Tone Multi-Frequency) keys with this screen. DTMF tones are used by many office phone systems ("Press 1 to talk with sales. Press 2 to talk with customer support.", etc.)

When you dial a digit, it will be immediately sent over the active phone call. When you have finished, click Done to exit this screen.


Setup HID (Top)
Sets up global HID-specific settings.

Enable Key Repeat enables or disables key-repeat for HID keyboards.

Delay: indicates the length of the delay between a key press and the first key repeat.

Rate: indicates the rate of key repetition (or typematic rate) when a key begins to repeat.
